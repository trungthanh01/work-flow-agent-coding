# Workflow Agent Coding — Second Brain Framework (Markdown-first)

Repo này là “khung vận hành” để AI agent (Cursor, Claude Code, …) giúp dev làm việc có hệ thống:
- **MOC-first**: luôn có “bảng chỉ đường” để agent không lạc
- **Problems → Lessons**: biến ca bệnh thành bài học + guardrails
- **Tôn trọng dữ liệu người dùng**: mặc định **Git không track** notes/logs của user (chỉ track framework)

## Bắt đầu nhanh
- Đọc `SETUP.md`
- Điểm vào cho agent: `00_moc/00_main-moc.md`

## Nguyên tắc update
Framework có thể update; dữ liệu người dùng không bị đụng (xem `PRD.md` mục “Cơ chế update framework”).

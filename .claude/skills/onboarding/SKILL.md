# Onboarding Skill

## Khi dùng
- User vừa clone repo và muốn “dùng được ngay”
- Thiếu `01_memory/MEMORY.md` hoặc các `INDEX.md`

## Mục tiêu
- Tạo cấu trúc content tối thiểu bằng script onboarding.

## Cách làm
1) Yêu cầu chạy script:

```bash
chmod +x scripts/*.sh
./scripts/onboard.sh
```

2) Sau đó nhắc user mở `00_moc/00_main-moc.md` để bắt đầu workflow.
